Formatting information for email templates

The email templates require some specific formatting in order to be pulled through properly.

Each paragraph should start with <p> and end with </p>. This makes sure that paragraphs are correctly spaced.

This site has the ability to automatically add information such as the project title, name of the PI and so on. The format for this is {{ variable_name }}
